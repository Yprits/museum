Серый. Переключатели все черные, расположены в один ряд. Технологический разъем и разъем для БРП присутствуют. Пломба черная, с оттиском "ОТК 10". В батарейном отсеке сохранился рулон полимерной пленки. На дисплее есть защитная пленка, немного налипла пыль. РЭ потрепаны, содержат список опечаток; часть 2 скреплена проволокой. Есть паспорт на БП Д2-37А. Продан 23.01.1990 ФМО "Электроника" г. Москва, секция № 2. Коробка оранжевая без одной стороны, с умеренными загрязнениями.
